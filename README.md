# Foliofy Portfolio

A modern, responsive personal portfolio website built with React, TypeScript, and Vite. 

## Features
- Dynamic and aesthetic user interface
- Smooth scrolling and animations (Framer Motion, Lenis)
- Fully responsive design using Tailwind CSS
- Iconography by Lucide React

## Technologies Used
- React 19
- TypeScript
- Vite
- Tailwind CSS (with tailwind-merge & clsx)
- Framer Motion
- React Router DOM

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

To start the local development server:
```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173/` (or the port specified in your terminal).

## Building for Production

To create an optimized production build:
```bash
npm run build
```

This will run TypeScript type-checking and build your application into the `dist` folder.
